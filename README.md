# Development

1. `npm install`
2. `npm run dev`
3. view at http://localhost:3000/

# Deploy

The first time you deploy, run `chmod +x deploy.sh` to ensure you have permission to run the script.

1. `npm run deploy`
2. that's it (only deploys to github pages)
