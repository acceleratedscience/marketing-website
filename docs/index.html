How the Discovery Platform works.Write applications anywhere with SciDK, our built-for-science developer kit.

Install SciDK and create a project.pip install scidk
scidk new 'project_name'Search public data and upload your own.scidk chemistry search --index_key pubmed 'keyword'
scidk data upload training-data.txt 'training_data'

Use a pre-trained model or train your own.scidk chemistry training_data train modelSet your target for compound generation.scidk data upload target.txt 'target'
